Boards allow you to organize tasks into customizable columns, and you can edit their details or delete them entirely from the sidebar menu. Editing updates board properties like the name without losing data, while deletion permanently removes the board along with all its columns and tasks—use the confirmation dialog to avoid accidental data loss. These actions help maintain a clean workspace as your needs evolve, and they integrate with the sidebar board list for quick access.

## Accessing Board Options Menu

The board options menu appears next to individual boards in the sidebar.

- Open the sidebar if it's hidden by clicking the **Hide Sidebar** button (expand icon) in the top-right of the sidebar panel.
- Locate the desired board in the **ALL BOARDS** list, which shows each board's icon, *name (column count)*.
- Hover over the board row (or select it to make it active, highlighted in primary color) to reveal the **more vertical** (three dots) icon on the right.
- Click the **more vertical** icon to open a popup menu with two options:
  | Option       | Description |
  |--------------|-------------|
  | **Edit Board** | Opens the edit modal to modify board details. |
  | **Delete Board** | Opens the delete confirmation dialog. |

> [!NOTE]  
> The menu only appears for the hovered or active board. See 3.1 Sidebar Board List for full sidebar navigation.

## Editing a Board

Use this to update the board's name or other properties. The process reuses the familiar board creation form, pre-filled with current values.

1. Hover over or select the board in the sidebar to show the **more vertical** icon.
2. Click the **more vertical** icon and select **Edit Board** from the popup menu.
3. A modal dialog opens, similar to the create board form (see 4.1 Creating a Board):
   - Fields are pre-populated with the current board's details (e.g., name).
   - Modify the fields as needed (required fields are marked; name accepts text strings).
4. Click **Save** or the equivalent confirm button in the modal.
5. The modal closes, and changes apply immediately—the updated board name and details appear in the sidebar and header.

Expected results:
- Sidebar refreshes to show the new name and column count.
- The board remains active unless you switch boards.
- No data loss occurs.

> [!TIP]  
> Changes sync across the app; no refresh needed.

## Deleting a Board

Deletion is irreversible and removes *all columns and tasks* within the board. A confirmation dialog ensures you intend this action.

1. Hover over or select the board in the sidebar to show the **more vertical** icon.
2. Click the **more vertical** icon and select **Delete Board** from the popup menu.
3. A confirmation modal opens with:
   - Header: **Delete *Board Name*** (e.g., "Delete Project Alpha").
   - Warning text: "Are you sure you want to delete this *board*? This action will remove all columns and tasks in this board."
   - Two buttons at the bottom:
     | Button     | Appearance          | Action |
     |------------|---------------------|--------|
     | **Delete** | Red background, hover darkens | Permanently deletes the board and closes the modal. |
     | **Cancel** | Gray background, hover darkens | Closes the modal without changes. |

4. Review the warning, then click **Delete** to confirm.
5. The modal closes, and the board is removed from the sidebar list (**ALL BOARDS** count decreases).

Expected results:
- All columns and tasks in the board are permanently deleted—no recovery option.
- The app switches to another board (if available) or the dashboard.
- Sidebar updates instantly.

> [!WARNING]  
> This action cannot be undone. Export tasks first if needed (future feature). Connects to 5. Managing Columns and 6. Creating and Managing Tasks—deleted boards remove related items.
