Deleting tasks permanently removes them from your board and column, freeing up space and helping you keep your workspace organized. This action is irreversible, so use it carefully for tasks that are no longer needed. Access this feature from the task details menu in any column.

## Accessing the Delete Option
To delete a task:
1. Open the board containing the task.
2. Locate the task card in its column.
3. Click the **menu icon** (three dots) on the task card to open the task details menu.
4. Select the **Delete** option from the menu.

This opens the **Delete Confirmation Dialog** overlay.

## The Delete Confirmation Dialog
This modal dialog appears centered on the screen with a dark overlay background. It confirms your intent before performing the deletion.

### Dialog Elements
| Element | Description |
|---------|-------------|
| **Title** | Displays "Delete *task name*" in bold red text at the top, where *task name* is the exact name of the selected task. |
| **Message** | Informational text reading: "Are you sure you want to delete this **task**? This action cannot be reversed." The word "task" is emphasized in bold and larger font. |

### Action Buttons
The buttons are centered at the bottom of the dialog:
- **Delete** button: Red-tinted background (appears as semi-transparent red), white text, bold font. Clicking this permanently deletes the task from the board and column, then closes the dialog automatically. The task card disappears immediately from the UI.
- **Cancel** button: Gray-tinted background (semi-transparent gray), bold font. Clicking this closes the dialog without any changes.

> [!WARNING]  
> Deleting a task removes it entirely—no recovery option exists. Double-check before confirming.

## Expected Results
- **Successful deletion**: The task vanishes from its column on the board. The board updates in real-time, and any related views refresh.
- **No changes on cancel**: The dialog closes, and the task remains unchanged.
