Viewing task details provides a quick way to review a task's key information, check subtask progress, toggle subtask completion, move the task to a different column, or access edit and delete options—all within a dedicated modal view. This is useful for monitoring progress and making minor adjustments without leaving the board or fully editing the task.

## Opening Task Details
To view details for any task:
1. Navigate to your active board via the sidebar board list (**3.1 Sidebar Board List**).
2. Locate the task card in its current column (**5. Managing Columns**).
3. Click anywhere on the task card (it shows the task title and subtask count, e.g., "*3 of 5 subtasks*").

A modal window opens, overlaying the board, displaying the full task details.

> [!NOTE]  
> Task cards are draggable between columns (**6. Creating and Managing Tasks**), but clicking opens details instead of dragging.

## Task Header
At the top of the modal:
- **Task title**: Displayed prominently in large, bold text (e.g., "Pending design task").
- **More options button** (vertical dots icon): Click to open a dropdown popup menu with these actions:

| Menu Option | Description | Result |
|-------------|-------------|--------|
| **Edit Task** | Opens the task editor modal. | Switches to the full edit view (**6.1 Adding Tasks** for editing workflow). |
| **Delete Task** | Confirms task deletion. | Shows a confirmation dialog with the task name; confirm to remove the task permanently from the board. |

## Description Section
- Below the header, a paragraph shows the task's description text.
- If no description is set, displays "*No description*".
- Text is in small, gray font for easy reading.

## Subtasks Section
- Labeled "**Subtasks (X of Y)**" in bold text, where *X* is completed subtasks and *Y* is total subtasks.
- Scrollable list (if 4+ subtasks, fixed height with scrollbar) of subtask items.
- Each subtask row includes:
  - **Checkbox**: Click to toggle completion (checked = complete).
    - Completed subtasks show struck-through text.
  - **Subtask title**: Plain text (e.g., "Add company logo").
- Changes to checkboxes update the count immediately and persist on the board.

> [!WARNING]  
> Subtasks must be added during task creation or editing (**7.2 Managing Subtasks**). You cannot add or edit subtasks here—use the full task editor.

## Column Selector
- At the bottom, labeled "**Column**" in bold text.
- A dropdown selector (**SelectBox**) shows the current column (e.g., "Todo") and lists all available columns on the board.
- Select a different column to move the task instantly.
  - Task closes and reappears in the new column at the top position.
  - No confirmation needed; change is immediate.

| Field | Description | Required? | Accepted Values | Default |
|-------|-------------|-----------|-----------------|---------|
| **Column** | Current or target column for the task. | Yes | Names of columns on the active board (e.g., "Todo", "In Progress", "Done"). | Task's current column. |

## Closing the View
- Click the **X** (close icon) in the modal header, press **Esc**, or click outside the modal.
- Changes (subtask toggles, column moves) save automatically.

## Related Features
- **Drag and drop**: Move tasks between columns directly on the board without opening details (**6. Creating and Managing Tasks**).
- **Full editing**: Use the **Edit Task** menu option or create new tasks with subtasks (**6.1 Adding Tasks**, **7.2 Managing Subtasks**).
- **Board customization**: Column names affect selector options (**5.2 Editing Column Names**).
