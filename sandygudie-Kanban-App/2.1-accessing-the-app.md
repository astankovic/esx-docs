The landing page is your entry point to the Kanban app, showcasing its key benefits like visual task management, drag-and-drop interactions, and multi-project support. It provides a welcoming overview with smooth animations and quick access to the dashboard workspace, allowing you to jump straight into creating and managing boards without any setup hurdles.

## Page Layout and Sections
The landing page features a full-screen design with a scenic background image, divided into scrollable sections for easy exploration:

- **Header**: Appears at the top with the app logo on the left (clickable to return to the landing page) and navigation links on the right for *Features*, *Technologies*, and *Contact*. These links smoothly scroll to their respective sections below.
- **Hero Section**: Centered title reading "Effortlessly Manage Your Projects" split across two lines, followed by a description highlighting visual task tracking, priority shifts, progress monitoring, and deadline management. Below it sits the prominent **Get Started** button.
- **Features Section**: Displays a screenshot of a full board view alongside a bulleted list of capabilities:
  | Feature | Description |
  |---------|-------------|
  | Simplified board view | Clean, intuitive layout for task organization. |
  | Handle multiple projects | Manage several boards simultaneously. |
  | Smooth drag and drop interaction | Easily move tasks between columns. |
  | Optimal layout on device's screen size | Responsive design adapts to any screen. |
  | Create, read, update, and delete boards and tasks | Full CRUD operations for all items. |
  | Keep track of any changes | History and updates are preserved. |
  | Toggle theme between dark and light mode | Switch UI themes as needed. |
  | Form validation | Ensures data entry is correct before saving. |
- **Technologies Section**: A scrolling marquee banner showing logos of tools used in the app, such as form libraries, animation tools, styling frameworks, state managers, type systems, and testing suites.
- **Connect Section**: Invitation to collaborate with a call-to-action text, followed by social icons linking externally:
  | Icon | Destination |
  |------|-------------|
  | GitHub | Opens the developer's GitHub profile in a new tab. |
  | LinkedIn | Opens the developer's LinkedIn profile in a new tab. |
  | Website | Opens the developer's portfolio site in a new tab. |
  | Email | Opens your default email client with a pre-filled address. |
- **Footer**: Simple copyright notice at the bottom, showing the current year and creator name.

## Key Actions and Interactions
- Click the **app logo** in the header to refresh or return to the landing page.
- Click navigation links (**Features**, **Technologies**, **Contact**) to scroll instantly to those sections.
- Hover over links and icons for subtle color or scale changes indicating interactivity.
- Click the **Get Started** button (blue, rounded, with white text) to navigate directly to the dashboard workspace. This transitions you to the main app area where you can create boards and start working—see 3. Dashboard and Navigation for details.
- Click social icons in the **Connect** section to open external profiles or your email app.

## Step-by-Step: Accessing the Dashboard
1. Open your web browser and navigate to the app's landing page URL.
2. Scroll through sections if desired to review features and technologies.
3. Locate the **Get Started** button in the hero section (or scroll up if needed).
4. Click **Get Started**—the page will load the dashboard view, displaying the sidebar for boards and the main workspace area.
5. If already logged in or on return visits, this action takes you straight to your workspace without prompts.

> [!NOTE]  
> The page uses smooth animations on load (fading in header, title, and description) for a polished feel, but these are visual only and do not affect functionality.

> [!WARNING]  
> External social links open in new tabs; ensure pop-up blockers allow this if links fail to load.

This page connects directly to the core app via the **Get Started** button, leading into board management and task workflows described in 4. Managing Boards and beyond. No account creation is needed initially—dive right into the dashboard.
